# LKB AVION
Le missionnaire met son nom.
Il indique si le trajet est en : a/ train ; b/ avion 2a/ Si train : indiquer le prix 2b/ Si avion : indiquer le CO2 (valeur estimée par l'interface Labo1.5). On collecte tout ça dans une base de donnée, pour garder trace de chaque mission et vérifier après coup s'il y a eu des problèmes (ne pas garder uniquement la somme).
